## 2️⃣ 학습목표

### ❌ 개선 전
```
[지식·이해] 지속가능한 미래를 저해하는 요인들을 이해할 수 있다.
[과정·기능] 지속가능한 미래의 저해 요인을 탐색하고 분석할 수 있다.
[가치·태도] 지속가능한 미래를 만들기 위한 변화의 책임에 공감할 수 있다.
```

### ✅ 개선 후
```
[지식·이해] 지속가능한 미래를 저해하는 요인들을 이해한다.
[과정·기능] 저해 요인을 탐색하고 분석하며 공정한 극복 방안을 제안한다.
[가치·태도] 공정한 기후 정의에 공감하고 변화의 책임을 실천한다.
```

**개선 이유**:
- "-한다" 형식으로 통일
- 핵심개념 "공정한 기후 정의" 반영
- 명확하고 능동적인 표현

## 3️⃣ 도입부

### ❌ 개선 전
```
▣ 동기유발
• "지속가능한 미래를 막는 것은 무엇일까요?" 발문
  - 학생들 자유 발표
  - 다양한 의견 수렴
• 저해 요인 탐색의 필요성
  - 문제를 알아야 해결 가능
  - 장애물을 파악해야 극복 가능

▣ 학습목표 제시
```

### ✅ 개선 후
```
▣ 환경과 삶 문제상황 제시
• **최근 COP 기후 협상 난항 뉴스**
  - "2023년 COP28에서 선진국과 개발도상국이 격렬히 대립했습니다"
  - 선진국: "모두 함께 탄소를 줄여야 한다"
  - 개발도상국: "역사적 책임은 선진국에 있다. 우리도 발전할 권리가 있다"
  - 결국 합의 지연...
• **발문**: "왜 전 세계가 모여도 합의하기 어려울까요?"
  - 학생들 자유 발표
  - "누구의 말이 맞을까요?"
  - "어떻게 하면 공정하게 해결할 수 있을까요?"
• **문제 인식**: 환경문제 해결을 방해하는 저해 요인들을 파악해야 극복 가능

▣ 학습목표 제시
```

**개선 이유**:
- 시사 기반 문제상황 제시 (COP 협상)
- 학생들의 관심과 호기심 유발
- 핵심 문제 (국가 간 이해 충돌) 담음
- 사고루틴 소개는 전개부로 이동

## 4️⃣ 전개부 (활동1)

### ❌ 개선 전
```
▣ [See] 저해 요인 사례 관찰
• 글로벌 저해 요인 탐색
  1) 화석연료 의존
     - 국가별 화석연료 의존도 통계
       · 중국: 석탄 발전 비중 높음
       · 사우디아라비아: 석유 경제 의존
       · 미국: 천연가스, 석유 생산 대국
     - "왜 화석연료를 계속 쓸까요?"
       · 경제적 이익 (일자리, 세수)
       · 기존 인프라 투자비
       · 전환 비용 부담
```

### ✅ 개선 후
```
▣ 사고루틴 소개: See-Think-Wonder
• "오늘은 보고-생각하고-궁금해하기 방법으로 탐구합니다"

▣ [See] 저해 요인 사례 관찰 (👨‍👩‍👧‍👦 모둠)
• **글로벌 저해 요인 탐색**
  1) 화석연료 의존
     - 국가별 화석연료 의존도 통계 관찰
       · 중국: 석탄 발전 비중 높음
       · 사우디아라비아: 석유 경제 의존
       · 미국: 천연가스, 석유 생산 대국
     - 토의: "왜 화석연료를 계속 쓸까요?"
     - 학생들이 경제적 이익, 인프라 투자, 전환 비용 등 발견

▣ [Think] 원인 분석 (👥 짝 → 🏫 전체)
• **국가 간 이해 충돌 탐색**
  - "왜 국제사회가 합의하기 어려울까?"
  - 짝과 토의할 수 있습니다:
    · 선택1: 각국의 이해관계 차이 분석하기
    · 선택2: 단기 이익과 장기 지속가능성 충돌 토론하기
  
• **역할극: 선진국 vs 개발도상국** (👨‍👩‍👧‍👦 모둠)
  - 모둠을 나누어 각 입장 대변할 수 있습니다:
    · 모둠1-2: 선진국 입장
    · 모둠3-4: 개발도상국 입장
    · 모둠5-6: 중재자 입장
  - 각 입장 발표
  - "합의가 왜 어려운지" 체감
```

**개선 이유**:
- 사고루틴을 전개부에서 소개
- 선택형 표현: "~할 수 있습니다. 선택1:___, 선택2:___"
- 학습 형태 아이콘 추가 (👤👥👨‍👩‍👧‍👦🏫)
- 구체적 시간 삭제
- "자원자" 표현 삭제

## 5️⃣ 정리부

### ❌ 개선 전
```
▣ 학습 정리
• 저해 요인 정리
  - 화석연료 의존, 국가 간 이해 충돌, 경제 우선주의, 소비 문화
• 극복 방안 정리
  - 공정한 기후 정의, 재생에너지 전환, 국제 협력 강화, 시민 참여
• "어렵지만 불가능하지 않다"
  - 이미 변화는 시작되었음
  - 우리의 참여가 중요
  - 함께하면 극복 가능

▣ 차시 예고
• 다음 시간: 블록 13 - 환경 감수성 기르기
• 자연과의 교감, 환경 감수성
```

### ✅ 개선 후
```
▣ 배움 성찰 및 자기 개념화 (👤 개인)
• **도입 문제상황 재연결**
  - "COP 협상에서 왜 합의가 어려웠을까요?" (도입 문제 회귀)
  - 학생들: 이제는 저해 요인들을 이해하고 설명할 수 있음

• **나의 배움 표현** - 다음 중 선택할 수 있습니다:
  - 선택1: 문장으로 정리하기
    · "공정한 기후 정의란 ___이다"
    · "저해 요인을 극복하려면 ___해야 한다"
  - 선택2: 그림으로 표현하기
    · 저해 요인과 극복 방안을 화살표로 연결
  - 선택3: 편지 쓰기
    · COP 협상가들에게 제안하는 편지

• **일반화 및 전이**
  - "어렵지만 불가능하지 않습니다"
  - 학생들의 표현을 교실에 전시
  - 이미 변화는 시작되었으며, 우리의 참여가 중요함을 공유

▣ 차시 예고
• 다음 시간: 블록 13 - 환경 감수성 기르기
```

**개선 이유**:
- 도입 문제상황(COP 협상)과 연결하여 순환 구조 완성
- 학생의 자기 개념화/일반화 활동 추가
- 다양한 표현 형식 제공 (문장/그림/편지)
- 지식 정리를 넘어 배움의 의미화

## 6️⃣ 학습지 정합성 검토

### 현재 학습지 구조
1. [See] 글로벌 저해 요인 - 표 작성

3. [Wonder] 극복 방안 - 표 작성
4. 나의 다짐

### 정합성 평가
✅ **정합성 유지됨**
- 지도안의 활동 순서와 학습지 구조 일치
- See-Think-Wonder 사고루틴 그대로 유지
- 역할극 활동이 학습지 2번 활동과 연결됨
- 극복 방안 탐색이 학습지 3번 활동과 연결됨

⚠️ **검토 필요**
- 정리부의 "나의 배움 표현" 활동은 현재 학습지에 없음
- 그러나 기존 "나의 다짐"과 통합 가능
- **학습지 수정 불필요** (최소 범위 유지)
