## 🔄 사고루틴 용어 한글화 목록

### 1. Bridge Routine → **다리 놓기 (브리지)**
- **설명**: 이전 프로젝트와 새 프로젝트를 연결
- **구조**: 처음 생각 → 새로운 경험 → 연결된 생각
- **사용 예**: "P1→P2 다리 놓기로 연결합니다"

### 2. Think-Puzzle-Explore → **생각-궁금증-탐색 (TPE)**
- **Think**: 생각하기 - 주제에 대해 알고 있는 것
- **Puzzle**: 궁금한 점 - 알고 싶은 것
- **Explore**: 탐색하기 - 어떻게 알아볼지 계획

### 3. Claim-Support-Question → **주장-근거-질문 (CSQ)**
- **Claim**: 주장 - 핵심 주장이나 정의
- **Support**: 근거 - 주장을 뒷받침하는 이유
- **Question**: 질문 - 더 알고 싶은 점

### 4. Compass Points → **나침반 사고 (방향 탐색)**
- **E (East/Excitement)**: 동쪽 - 기대되는 점
- **W (West/Worries)**: 서쪽 - 걱정되는 점
- **N (North/Needs)**: 북쪽 - 필요한 것
- **S (South/Suggestions)**: 남쪽 - 제안하고 싶은 것

### 5. Circle of Viewpoints → **관점 돌아보기 (다양한 시각)**
- **설명**: 여러 입장에서 생각해보기
- **구조**: 각 입장 → 그 입장의 생각 → 이유
- **예**: "학생, 교사, 학부모 관점 돌아보기"

### 6. Headlines → **헤드라인 (핵심 정리)**
- **설명**: 배운 내용을 신문 제목처럼 한 문장으로
- **목적**: 핵심 아이디어 일반화

### 7. Connect-Extend-Challenge → **연결-확장-도전 (CEC)**
- **Connect**: 연결 - 내 경험과 연결되는 것
- **Extend**: 확장 - 생각이 확장된 것
- **Challenge**: 도전 - 더 알고 싶은 것

### 8. See-Think-Wonder → **보기-생각하기-궁금해하기 (STW)**
- **See**: 보기 - 관찰한 것
- **Think**: 생각하기 - 든 생각
- **Wonder**: 궁금해하기 - 궁금한 점

### 9. Step Inside → **입장 바꿔보기 (역지사지)**
- **설명**: 다른 사람의 입장이 되어보기
- **활용**: 역할극, 공감 훈련

### 10. I Used to Think... Now I Think → **처음 생각...지금 생각 (변화 성찰)**
- **Used to Think**: 처음에 생각했던 것
- **Now I Think**: 지금 생각하는 것
- **목적**: 학습으로 인한 생각 변화 확인

### 11. What? So What? Now What? → **무엇을? 그래서? 이제는? (WSN)**
- **What**: 무엇을 - 배운 것
- **So What**: 그래서 - 왜 중요한지
- **Now What**: 이제는 - 어떻게 적용할지

### 12. Creative Questions → **창의적 질문하기**
- **설명**: "만약 ~라면?", "어떻게 하면?" 등 창의적 질문
- **목적**: 새로운 아이디어 발상

## 📝 파일 수정 방법

### 변경 전:
```markdown
#### 도입 (5분) - Bridge Routine
P1→P2 연결
```

### 변경 후:
```markdown
#### 도입 (5분) - 다리 놓기 (브리지)
**P1→P2 연결하기**:
- 처음 생각: "정체성은 나만의 특성"
- 경험: "우리 반 다양성 발견"
- 연결된 생각: "차이가 갈등으로?"
```
