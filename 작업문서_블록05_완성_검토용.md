# 블록5 완성본 검토 (Phase 5-6 적용)

## ✅ 적용된 개선 사항

### 1. 학습목표 개선
**기존**: "~을 이해할 수 있다"
**개선**: 
- [지식·이해] 기후변화의 원인과 전 지구적 영향을 **이해한다**.
- [과정·기능] **글로벌** 기후변화 현상을 관찰하고 원인과 결과를 **분석한다**.
- [가치·태도] 기후변화 해결에 동참하려는 마음을 가지고 **실천한다**.

---

### 2. 개념 구조 개선
**기존**: 
```
관련개념: 기후변화, 온실가스, 지구온난화, IPCC, 1.5도 목표, 글로벌 영향
```

**개선**:
```
핵심개념: 기후변화의 글로벌 영향
추가개념: 온실가스와 지구온난화, IPCC 1.5도 목표, 국가별 기후 영향
```

---

### 3. 도입부 문제상황
**기존**: 
- 날씨와 기후 구분하기 (개념 설명 중심)
- 학생 경험 나누기

**개선**:
```
▣ 환경과 삶 문제상황 제시
• **지구의 체온이 올라가고 있다**
  - 뉴스 클립: 2023년 여름, 역대 최고 기온
    · 대구 40도 폭염, 서울 열대야 연속
    · 기상캐스터: "지구 평균 기온이 계속 상승"
  
  - **세계 곳곳의 이상 기후** (사진 4장 동시 제시)
    1. 북극: 빙하가 녹아 떨어지는 장면
    2. 투발루: 섬이 물에 잠기는 모습
    3. 방글라데시: 홍수로 집이 떠내려감
    4. 아프리카: 가뭄으로 갈라진 땅

• **발문**
  - "이 네 장의 사진, 무엇이 공통점일까요?"
  - "왜 지구 곳곳에서 이런 일이 일어날까요?"
```

**개선 효과**:
- 시사 기반 구체적 문제상황
- 글로벌 차원 문제 명확히 제시
- 긴급성과 심각성 인식

### 4. 전개부 개선

#### 4-1. 사고루틴 소개 위치 변경 ✅
**개선**: 전개부에서 직접 소개

#### 4-2. 선택형 표현 적용 ✅
```
• 세계 지도 표시 활동 (👨‍👩‍👧‍👦 모둠)
  - 다음 중 선택할 수 있습니다:
    · 선택1: 각 지역을 세계 지도에 표시하고 영향 정리하기
    · 선택2: 가장 심각한 지역 3곳 선정하고 이유 설명하기

• 모둠별로 다음 중 선택할 수 있습니다:
    · 선택1: 해당 지역의 기후변화 영향과 주민들의 삶 조사하기
    · 선택2: 국제사회의 지원 방안과 협력 사례 찾아보기
```

#### 4-3. 학습 형태 아이콘 추가 ✅
```
[See] 글로벌 기후변화 현상 관찰 (👨‍👩‍👧‍👦 모둠 → 🏫 전체)
[Wonder] 기후변화가 삶에 미치는 영향 탐색 (👨‍👩‍👧‍👦 모둠 → 🏫 전체)
"우리나라는 어떤 영향을 받고 있을까요?" (👥 짝 → 🏫 전체)
탄소발자국 측정 앱 체험 (👤 개인)
```

#### 4-4. 구체적 시간/인원 삭제 ✅
**기존**: "(25분)", "(20분)", "(5분)", "각 3분"
**개선**: 시간 칸 비우기

#### 4-5. 차별화 강조 유지 ⭐
```
※ **차별화 준수: 과학 원리 3-5분**
※ **글로벌 영향에 집중 (25분)**
```
- 유의점에 강조 표시
- 블록5의 특징(차별화) 명확히 유지

### 5. 정리부 개선 (순환 구조 완성)

**기존**:
```
▣ 학습 정리
• 기후변화 원인-결과-글로벌 영향 정리
• 퀴즈: 국가별 온실가스 배출 순위
• "우리도 책임이 있다" 인식
```

**개선**:
```
▣ 배움 성찰 및 자기 개념화 (👤 개인)
• **도입 문제상황 재연결**
  - "지구 곳곳에서 왜 이상 기후가 일어날까요?" (도입 문제 회귀)
  - 🏫 학생들: 이제는 온실가스, IPCC 1.5도 목표, 글로벌 영향 설명 가능
  - "북극, 투발루, 방글라데시, 아프리카 모두 연결"

• **나의 배움 표현** - 다음 중 선택:
  - 선택1: 문장으로 정리하기
    · "기후변화의 가장 큰 영향을 받는 곳은 ___이고, 이유는 ___이다"
    · "지구 온도 1.5도 상승이 중요한 이유는 ___이다"
  - 선택2: 그림으로 표현하기
    · 세계 지도에 기후변화 영향을 화살표와 그림으로 표시
  - 선택3: 편지 쓰기
    · 기후변화로 고통받는 지역 사람들에게 연대의 편지

• **국가별 온실가스 배출 순위** (퀴즈 형식)
  - 1위: 중국, 2위: 미국, 3위: 인도
  - 한국: 세계 7-8위 (인구 대비 높은 편)
  - "우리도 책임이 있습니다"

• **일반화 및 전이**
  - "기후변화는 전 세계 모든 사람의 문제입니다"
  - "우리의 작은 행동이 지구 반대편 사람들의 삶에 영향을 미칩니다"
```

**개선 효과**:
- 도입 문제(이상 기후 4장 사진) → 정리에서 재연결
- 글로벌 연대감 강조
- 퀴즈를 성찰의 일부로 통합
- 다양한 표현 형식 제공
