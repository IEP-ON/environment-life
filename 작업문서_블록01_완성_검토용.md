# 블록1 완성본 검토 (Phase 5-6 적용)
## 오프닝 세션 특수성 반영

## ✅ 적용된 개선 사항

### 1. 학습목표 개선
**기존**: "~을 이해할 수 있다"
**개선**: 
- [지식·이해] 일상생활과 환경의 글로벌 연결성을 **이해한다**.
- [과정·기능] 나의 삶과 세계 환경이 연결되어 있음을 **파악한다**.
- [가치·태도] 환경에 대한 궁금증을 가지고 탐구하려는 **태도를 기른다**.

---

### 2. 개념 구조 개선
**기존**: 
```
관련개념: 환경, 일상생활, 연결성, 글로벌, 세계화
```

**개선**:
```
핵심개념: 글로벌 연결성
추가개념: 일상과 환경 연결, 세계화, 환경 문제의 상호의존성
```

---

### 3. 도입부 문제상황 (오프닝 특성 반영)
**기존**: 
- "오늘 아침부터 사용한 것들을 떠올려 볼까요?"
- 일반적인 질문

**개선**:
```
▣ 환경과 삶 문제상황 제시
• **나의 아침 루틴과 세계**
  - 화면 제시: 한 학생의 아침 풍경 사진
    · 침대에서 일어나 스마트폰 확인 (중국 제조)
    · 티셔츠 입기 (베트남 생산)
    · 바나나 먹기 (필리핀산)
    · 가방 메고 등교 (인도네시아 제조)

• **발문**: "여러분의 아침은 어땠나요?"
  - 사용한 물건들을 떠올려보기
  - "이 물건들은 어디서 왔을까요?"

• **세계 지도 활동** (🏫 전체)
  - 학생들이 가진 물건의 라벨 확인
  - 지도에 표시하기
  - 선으로 연결하면 → 전 세계와 연결된 그물망

• **핵심 질문**
  - "우리는 알게 모르게 이미 전 세계와 연결되어 있습니다"
  - "환경은 어떨까요? 우리의 일상이 지구 환경과도 연결되어 있을까요?"
  - "32차시 동안 함께 탐구해봅시다!"

▣ 32차시 여정 안내
• 오늘부터 시작되는 배움의 여정
  - 영역1: 세계화와 상호의존 (블록 1-7)
  - 영역2: 생태전환 (블록 9-16)
  - 클로징: 블록 17
```

**개선 효과**:
- 구체적인 아침 루틴 시나리오 제공
- 글로벌 연결성 시각화
- 32차시 전체 여정 소개 (오프닝 역할)

---

### 4. 전개부 개선 (오프닝 특성 유지)

#### 4-1. 사고루틴 소개 ✅
```
▣ 사고루틴 소개: See-Think-Wonder
• "오늘은 보고-생각하고-궁금해하기 방법으로 탐구합니다"
• "32차시 동안 이 방법을 자주 사용할 거예요"
```
- 오프닝답게 사고루틴 처음 소개
- 32차시 전체에서 사용할 것임을 안내

#### 4-2. 선택형 표현 적용 ✅
```
• 모둠 활동 - 다음 중 선택할 수 있습니다:
  - 선택1: 가장 인상 깊은 사진 3장 선택하고 이유 말하기
  - 선택2: 긍정적 사진과 부정적 사진 분류하고 느낌 나누기
```

#### 4-3. 학습 형태 아이콘 추가 ✅
```
[See] 환경 사진 관찰하기 (👨‍👩‍👧‍👦 모둠 → 🏫 전체)
[Think] 환경과 나의 연결고리 (👤 개인 → 👥 짝 → 🏫 전체)
[Wonder] 궁금한 점 나누기 (👨‍👩‍👧‍👦 모둠 → 🏫 전체)
```
- Think-Pair-Share 단계별 아이콘 명시

#### 4-4. 질문 벽 만들기 (블록1 고유 활동 유지) ⭐
```
▣ [Wonder] 궁금한 점 나누기
• 모둠별 궁금증 목록 작성
  - "환경에 대해 궁금한 점은 무엇인가요?"
  - 포스트잇에 질문 작성
• 질문 벽(Question Wall) 만들기
  - 모둠별 질문을 교실 질문 벽에 붙이기
  - 전체 질문 함께 읽기
• "이 질문들은 32차시 동안 함께 해결해 나갈 거예요"
```
- **블록1의 핵심 활동 유지**
- 32차시 전체와 연결되는 중요한 장치

---

### 5. 정리부 개선 (32차시 여정 시작 강조)

**기존**:
```
▣ 학습 정리
• 오늘 배운 내용 정리
  - 우리는 일상생활에서 이미 세계와 연결
  - 나의 작은 선택이 환경에 영향
  - 환경은 지역적이면서 동시에 글로벌한 문제
• 학습 소감 나누기 (자원자 3명)
```

**개선**:
```
▣ 배움 성찰 및 32차시 여정 시작 (👤 개인)
• **도입 문제상황 재연결**
  - "우리의 일상이 지구 환경과 연결되어 있을까요?" (도입 질문 회귀)
  - 🏫 학생들: 이제는 글로벌 연결성을 이해하고 설명할 수 있음
  - "물건도, 환경도 모두 세계와 연결되어 있다"

• **나의 배움 표현** - 다음 중 선택:
  - 선택1: 문장으로 정리하기
    · "오늘 가장 인상 깊었던 것은 ___이다"
    · "나의 ___행동이 ___에 영향을 미친다"
  - 선택2: 그림으로 표현하기
    · 나의 일상과 지구 환경의 연결고리를 화살표로 표시
  - 선택3: 질문 벽에서 가장 궁금한 질문 선택하고 이유 쓰기

• **32차시 여정의 시작** ⭐
  - "오늘은 32차시 여정의 첫걸음입니다"
  - "질문 벽의 질문들을 하나씩 해결하며 함께 성장해갑시다"
  - 학생들의 표현을 교실에 전시
  - 질문 벽은 32차시 동안 계속 유지
```

**개선 효과**:
- 도입 문제(아침 루틴) → 정리에서 재연결
- 자기 개념화 3가지 선택
- **32차시 여정 시작 강조** (오프닝 고유 역할)
- 질문 벽을 32차시 동안 유지하는 장치
