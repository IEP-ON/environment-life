## 📝 최종 산출물

### 📄 1. 지도안(약안, HTML 형식) - 블록당 1개

#### ✅ 형식 및 분량
- **파일 형식**: HTML (인쇄 시 A4 최적화)
- **분량**: A4 1-2페이지 이내
- **스타일**: 약안 샘플 HTML 형식 준수 + 공통 CSS 링크 필수(`assets/css/base.css`, `assets/css/print.css`), `body class="plan"`
- **글꼴**: Malgun Gothic, 화면 11pt / 인쇄 9.5–10pt (print.css 적용)
- **여백**: 10mm (상하좌우, print.css 적용)
- **인쇄 설정(브라우저)**: Headers/Footers Off, Background graphics On, Scale 100% (필요 시 95–90%)

#### ✅ 필수 구성 요소

**1) 기본 정보 테이블**
```
- 블록 번호 (뱃지 스타일)
- 일시, 장소, 대상
- 수업자, 단원, 차시, 주제
```

**2) 학습 설계 테이블**
```
- 성취기준 (전체 기재)
- 학습목표 (3개 이내, • 불렛 사용)
- 핵심개념 (5개 이내)
- 사고루틴 (루틴명 + 활용단계)
- 학습모형
```

**3) 교수학습 자료 테이블** ⭐ 중요
```
4열 구성:
① 텍스트화 자료 활용
   - 텍스트화_자료 폴더의 실제 파일명 명시
   - 예: • 초등_1-2차시(전기에너지).txt
        • 교사용_초등5~6학년-1-2차시.txt

② 교구/자료
   - 수업에 필요한 구체적 자료
   - 예: • 환경 사진 카드, 포스트잇, 실천 다짐판

③ 멀티미디어
   - 구체적인 출처와 제목 명시 (할루시네이션 금지)
   - 예: • EBS '지구의 경고' 영상 (3분)
        • 유네스코 SDGs 소개 영상
        • 그레타 툰베리 유엔 연설 영상
   - 온라인 자료의 경우 기관명 명시

④ 디지털 도구
   - 예: • 패들렛(Padlet), 멘티미터, 카훗, 구글 제미스턴
```

**4) 교수·학습 활동 테이블** ⭐ 핵심

   - ▣ 대주제 (예: ▣ 동기유발, ▣ [See] 환경문제 관찰)
