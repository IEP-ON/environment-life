# 블록12 완성본 검토 (Phase 5-6 적용)

## ✅ 적용된 개선 사항

### 1. 학습목표 개선
**기존**: "~을 이해할 수 있다"
**개선**: 
- [지식·이해] 지속가능한 미래를 저해하는 요인들을 **이해한다**.
- [과정·기능] 저해 요인을 탐색하고 분석하며 **공정한 극복 방안을 제안한다**.
- [가치·태도] **공정한 기후 정의**에 공감하고 변화의 책임을 **실천한다**.

---

### 2. 개념 구조 개선
**기존**: 
```
관련개념: 저해 요인, 화석연료 의존, 국가 간 이해 충돌, 경제 vs 환경, 극복 방안
```

**개선**:
```
핵심개념: 공정한 기후 정의
추가개념: 화석연료 의존, 국가 간 이해 충돌, 재생에너지 전환
```

---

### 3. 도입부 문제상황
**기존**: 
- "지속가능한 미래를 막는 것은 무엇일까요?" 발문
- 일반적인 질문

**개선**:
```
▣ 환경과 삶 문제상황 제시
• **최근 COP 기후 협상 난항 뉴스**
  - "2023년 COP28에서 선진국과 개발도상국이 격렬히 대립"
  - 선진국: "모두 함께 탄소를 줄여야 한다"
  - 개발도상국: "역사적 책임은 선진국에 있다. 우리도 발전할 권리"
  - 결국 합의 지연...
• **발문**: "왜 전 세계가 모여도 합의하기 어려울까요?"
  - 🏫 학생들 자유 발표
  - "누구의 말이 맞을까요?"
  - "어떻게 하면 공정하게 해결할 수 있을까요?"
```

**개선 효과**:
- 시사 기반 구체적 문제상황
- 핵심 문제(국가 간 이해 충돌) 명확히 제시
- 학생 흥미 유발

---

### 4. 전개부 개선

#### 4-1. 사고루틴 소개 위치 변경
**기존**: 도입부나 성취기준 표에만 명시
**개선**: 전개부에서 직접 소개
```
▣ 사고루틴 소개: See-Think-Wonder
• "오늘은 보고-생각하고-궁금해하기 방법으로 탐구합니다"
```

#### 4-2. 선택형 표현 적용
**기존**: "짝과 토의한다", "모둠별 발표 (각 3분)"
**개선**:
```
• 짝과 토의할 수 있습니다:
  · 선택1: 각국의 이해관계 차이 분석하기
  · 선택2: 단기 이익과 장기 지속가능성 충돌 토론하기

• 다음 방식 중 선택할 수 있습니다:
  · 선택1: 가장 효과적인 방안 3가지 선정하기
  · 선택2: 실현 가능성 순위 매기기
  · 선택3: 우리가 할 수 있는 것부터 분류하기
```

#### 4-3. 학습 형태 아이콘 추가
**추가된 아이콘**:
- 👤 개인
- 👥 짝
- 👨‍👩‍👧‍👦 모둠
- 🏫 전체

**적용 예시**:
```
▣ [See] 저해 요인 사례 관찰 (👨‍👩‍👧‍👦 모둠)
▣ [Think] 원인 분석 (👥 짝 → 🏫 전체)
▣ [Wonder] 극복 방안 탐구 (👨‍👩‍👧‍👦 모둠 → 🏫 전체)
```

#### 4-4. 구체적 시간/인원 삭제
**기존**: "각 3분", "자원자 5명 발표"
**개선**: 시간 칸 비우기, "발표" (인원 표시 없음)

---

### 5. 정리부 개선 (순환 구조 완성)

**기존**:
```
▣ 학습 정리
• 저해 요인 정리: 화석연료 의존, 국가 간 이해 충돌...
• 극복 방안 정리: 공정한 기후 정의, 재생에너지 전환...
• "어렵지만 불가능하지 않다"

▣ 차시 예고
```

**개선**:
```
▣ 배움 성찰 및 자기 개념화 (👤 개인)
• **도입 문제상황 재연결**
  - "COP 협상에서 왜 합의가 어려웠을까요?" (도입 문제 회귀)
  - 🏫 학생들: 이제는 저해 요인들을 이해하고 설명할 수 있음
  - "역사적 책임과 현재 책임", "공정한 해결" 개념 확립

• **나의 배움 표현** - 다음 중 선택할 수 있습니다:
  - 선택1: 문장으로 정리하기
    · "공정한 기후 정의란 ___이다"
    · "저해 요인을 극복하려면 ___해야 한다"
  - 선택2: 그림으로 표현하기
    · 저해 요인과 극복 방안을 화살표로 연결
  - 선택3: 편지 쓰기
    · COP 협상가들에게 제안하는 편지

• **일반화 및 전이**
  - "어렵지만 불가능하지 않습니다"
  - 학생들의 표현을 교실에 전시
  - 이미 변화는 시작되었으며, 우리의 참여가 중요함을 공유

▣ 차시 예고
```

**개선 효과**:
- 도입 문제(COP 협상) → 정리에서 재연결 (순환 구조)
- 지식 정리 → 배움의 의미화
- 학생 주도 자기 개념화
- 다양한 표현 형식 제공

## 🔍 학습지 정합성 확인

### 현재 학습지 구조 (블록12)
1. [See] 글로벌 저해 요인 - 표 작성
2. [Think] 선진국 vs 개발도상국 - 입장 비교표
3. [Wonder] 극복 방안 - 표 작성
4. 나의 다짐

### 정합성 평가
✅ **완벽히 일치**
- 지도안 활동 순서 = 학습지 구조
- See-Think-Wonder 사고루틴 유지
- 역할극 활동 → 학습지 2번과 연결
- 극복 방안 탐색 → 학습지 3번과 연결

⚠️ **정리부 추가 활동**
- "나의 배움 표현" 활동은 현재 학습지에 없음
- 그러나 "나의 다짐"과 통합 가능
- **학습지 수정 불필요** (최소 범위 유지 원칙)
