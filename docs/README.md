# 환경과 삶 II 교육과정 개발 보고서

## 📁 파일 구조

```
report/
├── index.html           # 메인 HTML 파일
├── styles.css           # 스타일시트
├── script.js            # 메인 JavaScript
├── blocks-data.js       # 17개 블록 데이터
├── script-helpers.js    # 추가 헬퍼 함수
└── README.md           # 이 파일
```

## 🚀 사용 방법

### 1. 보고서 열기
- `index.html` 파일을 더블클릭하거나
- 웹 브라우저로 드래그하여 열기

### 2. 주요 기능
- **네비게이션**: 상단 메뉴로 각 섹션 이동
- **블록 필터링**: 17개 블록을 카테고리별로 필터링
- **지도안·학습지 연동**: 각 블록 카드에서 바로 파일 열기
- **반응형 디자인**: PC, 태블릿, 모바일 모두 지원

### 3. 섹션 구성
1. **프로젝트 개요** - 교육과정 철학 및 성취기준
2. **교육과정 구성** - 32차시 블록타임 구조
3. **블록수업 상세** - 17개 블록별 상세 정보 및 파일 링크
4. **개발 자료 현황** - 지도안·학습지 현황
5. **특징 및 차별점** - 타 교과 차별화 전략
6. **향후 계획** - 현장 적용 및 확산 방안

## 📊 블록 데이터

### 17개 블록 구성
- **블록 1**: 오프닝 (1차시)
- **블록 2-7**: 영역1 - 세계화와 상호의존 (12차시)
- **블록 8**: 총괄평가 I (2차시)
- **블록 9-15**: 영역2 - 생태전환 (14차시)
- **블록 16**: 총괄평가 II (2차시)
- **블록 17**: 클로징 (1차시)

### 각 블록 정보
- 차시 번호
- 학습 주제
- 학습 목표
- 사고루틴
- 핵심 키워드
- 지도안 파일 링크
- 학습지 파일 링크

## 🔗 파일 연동

### 지도안 폴더
```
../지도안/블록XX_지도안_주제.html
```

### 학습지 폴더
```
../학습지/블록XX_학습지_주제.html
```

모든 링크는 상대 경로로 연결되어 있어 폴더 이동 시에도 작동합니다.
