> **세계화 시대, 환경문제를 글로벌 관점에서 이해하고 세계시민으로서의 환경 책임을 실천하는 교육과정**

## 🌐 온라인 보고서 접속

### GitHub Pages 배포 (권장)
```
https://iep-on.github.io/environment-life/
```

### 로컬 실행
1. 이 저장소를 클론합니다
2. `docs/index.html`을 웹 브라우저로 엽니다
3. 또는 로컬 서버 실행:
   ```bash
   cd docs
   python3 -m http.server 8000
   # 브라우저에서 http://localhost:8000 접속
   ```
