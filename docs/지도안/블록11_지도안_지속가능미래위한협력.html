평가 내용(관점) 평가 기준 평가 방법
문제 특성과 미래의 관련성을 추론하고 4개 차원의 협력 방안을 제안할 수 있는가? 상: 문제 특성과 지속가능한 미래의 관련성을 명확히 추론하고, 국제 협력의 중요성을 이해하며, 4개 차원의 구체적이고 실천 가능한 협력 방안을 제안함 • 관찰평가 • 산출물평가 (협력 계획) • 자기평가
중: 문제 특성과 미래의 관련성을 이해하고, 협력 방안을 제안함
하: 협력의 필요성을 인식하고, 협력 방안 수립에 참여함
