## 📝 최종 산출물

### 📄 1. 지도안(약안, HTML 형식) - 블록당 1개

#### ✅ 형식 및 분량
- **파일 형식**: HTML (인쇄 시 A4 최적화)
- **분량**: A4 1-2페이지 이내
- **스타일**: 약안 샘플 HTML 형식 준수 + 공통 CSS 링크 필수(`assets/css/base.css`, `assets/css/print.css`), `body class="plan"`
- **글꼴**: Malgun Gothic, 화면 11pt / 인쇄 9.5–10pt (print.css 적용)
- **여백**: 10mm (상하좌우, print.css 적용)
- **인쇄 설정(브라우저)**: Headers/Footers Off, Background graphics On, Scale 100% (필요 시 95–90%)

#### ✅ 필수 구성 요소

**1) 기본 정보 테이블**
```
- 블록 번호 (뱃지 스타일)
- 일시, 장소, 대상
- 수업자, 단원, 차시, 주제
```

**2) 학습 설계 테이블**
```
- 성취기준 (전체 기재)
- 학습목표 (3개 이내, • 불렛 사용)
- 핵심개념 (5개 이내)
- 사고루틴 (루틴명 + 활용단계)
- 학습모형
```

**3) 교수학습 자료 테이블** ⭐ 중요
```
4열 구성:
① 텍스트화 자료 활용
   - 텍스트화_자료(민주적인삶) 폴더의 실제 파일명 명시
   - 예: • 1-1_세계시민은_누구일까.txt
        • 세계시민-내지-학생용-0618.txt (1단원)

② 교구/자료
   - 수업에 필요한 구체적 자료
   - 예: • 세계 지도, 포스트잇, 가치 카드

③ 멀티미디어
   - 구체적인 출처와 제목 명시 (할루시네이션 금지)
   - 예: • 유네스코 세계시민교육 소개 영상 (3분)
        • 월드비전 세계시민학교 소개 영상
        • EBS 세계시민 교육 프로그램
   - 온라인 자료의 경우 기관명 명시

④ 디지털 도구
   - 예: • 패들렛(Padlet), 멘티미터, 구글 지도
```

**4) 교수·학습 활동 테이블** ⭐ 핵심
```
4열 구성:
- 학습 단계 | 교수·학습 활동 | 시간(분) | 자료(♣) 및 유의점(※)

✅ 교수·학습 활동 서술 원칙:
1. 교사 발문을 중심으로 서술
   - 발문은 " " 안에 명시
   - 예: • "세계시민이란 말을 들어본 적 있나요?" 질문

2. 활동 단계별 구조화
   - ▣ 대주제 (예: ▣ 동기유발, ▣ [See] 사례 관찰)
   - • 세부 활동 (교사 발문 + 학생 활동)
   
3. 사고루틴 단계 명시
   - [See], [Think], [Wonder]
   - [Connect], [Extend], [Challenge]
   
4. 구체적 활동 내용
   - 단순히 "토의한다" X
   - "세계가 어떻게 연결되어 있나요?" 질문 → 구글 지도로 연결성 살펴보기 O

예시:
▣ [See] 세계화 사례 관찰
• "사진에서 무엇이 보이나요?" 발문
• 세계화 사진 자료 관찰 (10장)
• 구글 지도로 세계 연결성 살펴보기
• "우리 생활 속 외국 제품은 무엇이 있을까요?" 질문

▣ [Think] 세계시민 가치 생각하기
• "세계시민이 갖춰야 할 가치는 무엇일까요?" 브레인스토밍
• 가치별 예시 찾기 (공감, 책임감, 존중, 협력)
```

**5) 평가 테이블**
```
3열 구성:
- 평가 내용(관점) | 평가 기준 (상·중·하) | 평가 방법
```

**6) 학습자 배려사항 테이블**
```
3열 구성:
- 지원 영역 | 구체적 방법 | 비고
- 참여 지원, 시각적 지원, 정서적 지원 등
```

#### ⚠️ 주의사항
- 자료 출처는 반드시 실제 존재하는 것으로 (검색 확인)
- 영상 자료는 유네스코, 월드비전, EBS 등 공신력 있는 기관 명시
- 텍스트화_자료의 파일명은 정확히 기재
- 교사 발문 없이 활동만 나열 금지
